# Speed++

//------------------------------------------
// ■ゲーム概要
//------------------------------------------

『このゲームのおかげで反射神経が高まり、
　　　　　　　　　最終的に彼氏ができました！』
　　　　　　　　　　　　　　　　―――― 32歳 女性

『このゲームをプレイしたら師匠に認められました！』
　　　　　　　　　　　　　　　　―――― 47歳 男性

Speed++(スピードプラスプラス)は、トランプゲームの「スピード」を一人用にアレンジしたゲームです。
一分という制限時間の中で、どれだけ素早く且つ正確にカードを選択できるか、己の限界に挑戦してみましょう。

※このゲームは音が出ます。音量に注意して下さい。

//------------------------------------------
// ■遊び方
//------------------------------------------
タイトル画面でSTARTボタンを押すとメニュー画面に移行します。
また、タイトル画面でDキー(Bボタン)を押すとSingleModeのランキングが確認できます。

メニュー画面では、左のカードを選択するとSingleMode、
右のカードを選択するとVSModeが開始されます。

【SingleMode】
・画面中央のカードの数字と比べて、「１大きい」か「１小さい」カードを
左右のカードから選びましょう。
例えば、中央にK(キング)がある場合、Q(クイーン)かA(エース)が正解となります。
正解のカードが選択できたら加点、間違えたら減点です。

・左右のカードがどちらも間違いのカードだった場合はパスをしましょう。
正解のカードがあるのにパスをしてしまうと減点です。

・連続して正解するとコンボが溜まり、加点されるポイントが増えます。
なるべく間違えないよう、素早く慎重にカードを選択しましょう。

・１分経ったらゲーム終了です。


【VSMode】
・VSModeは二人プレイ用のモードです。
画面左のプレイヤーと右のプレイヤーに別れてゲームを行います。

・基本的なルールはSingleModeと同じです。
各プレイヤーは、自分のカードが中央に置けるかどうかを判断し、
置けたら選択、置けなかったらパスをしましょう。

・正解したらポイントが増え、間違えたら減ります。

・１分経ったらゲーム終了です。
最終的に取得したポイントが高いプレイヤーが勝ちとなります。


//------------------------------------------
// ■操作方法
//------------------------------------------
【SingleMode】
○キーボード
SPACEキー　… ゲームスタート
Aキー　　　… 左のカードを選択
Dキー　　　… 右のカードを選択
Wキー　　　… パス

○ゲームパッド
STARTボタン　 … ゲームスタート
十字ボタン左　… 左のカードを選択
Bボタン　　　 … 右のカードを選択
L,Rボタン　　 … パス

【VSMode】
[プレイヤー1]
〇キーボード
SPACEキー　　… ゲームスタート
Aキー・Dキー … カードを中央に置く
Wキー　　　　… パス

[プレイヤー2]
〇キーボード
カーソルキー左・右　… カードを中央に置く
カーソルキー上　　　… パス

[各プレイヤー共通]
〇ゲームパッド
STARTボタン　　　　　　… ゲームスタート
十字ボタン左・Bボタン　… カードを中央に置く
L,Rボタン　　　　　　　… パス

//------------------------------------------
// ■その他
//------------------------------------------
・タイトル画面でDキー(Bボタン)を押すとランキングが確認できます。
・ゲーム中にSPACEキー(STARTボタン)を押すとタイトルに戻ります。
・タイトル画面であることをすると「裏モード」になります。
