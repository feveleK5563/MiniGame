# Speed++

//------------------------------------------
// ■ゲーム概要
//------------------------------------------

『このゲームのおかげで反射神経が高まり、
　　　　　　　　　最終的に彼氏ができました！』
　　　　　　　　　　　　　　　　―――― 32歳 女性

『このゲームをプレイしたら師匠に認められました！』
　　　　　　　　　　　　　　　　―――― 47歳 男性

Speed++(スピードプラスプラス)は、トランプゲームの「スピード」を一人用にアレンジしたゲームです。
一分という制限時間の中で、どれだけ素早く且つ正確にカードを選択できるか、己の限界に挑戦してみましょう。


//------------------------------------------
// ■遊び方
//------------------------------------------
・画面中央のカードの数字と比べて、「１大きい」か「１小さい」カードを
左右のカードから選びましょう。
例えば、中央にK(キング)がある場合、Q(クイーン)かA(エース)が正解となります。
正解のカードが選択できたら加点、間違えたら減点です。

・左右のカードがどちらも間違いのカードだった場合はパスをしましょう。
正解のカードがあるのにパスをしてしまうと減点です。

・連続して正解するとコンボが溜まり、加点されるポイントが増えます。
なるべく間違えないよう、素早く慎重にカードを選択しましょう。

・１分経ったらゲーム終了です。


//------------------------------------------
// ■操作方法
//------------------------------------------
○キーボード
SPACEキー　… ゲームスタート
Aキー　　　… 左のカードを選択
Dキー　　　… 右のカードを選択
Wキー　　　… パス

○ゲームパッド
STARTボタン　 … ゲームスタート
十字ボタン左　… 左のカードを選択
Bボタン　　　 … 右のカードを選択
L,Rボタン　　 … パス


//------------------------------------------
// ■その他
//------------------------------------------
・タイトル画面でDキー(Bボタン)を押すとランキングが確認できます。
・ゲーム中にSPACEキー(STARTボタン)を押すとタイトルに戻ります。
・タイトル画面であることをすると「裏モード」になります。
